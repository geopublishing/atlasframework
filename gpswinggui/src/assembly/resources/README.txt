This file describes how to start the Geopublisher application version ${project.version}-r${buildNumber}.     

* To start Geopublisher on WINDOWS:
1. Change into the Geopublisher-${project.version} directory 
2. run Geopublisher.exe

* To start Geopublisher on LINUX/MACOS:
1. Ensure that you have at least Java 6 JRE or later installed. Run 'java -version' to see your Java version.
2. Change into the Geopublisher-${project.version} directory 
4. ./start_Geopublisher.sh
