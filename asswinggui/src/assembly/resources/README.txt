This file describes how to start the AtlasStyler SLD Editor application version ${project.version}-r${buildNumber}.     

* To start AtlasStyler on WINDOWS:
1. Change into the AtlasStyler-${project.version} directory 
2. run AtlasStyler.exe

* To start AtlasStyler on LINUX/MACOS:
1. Ensure that you have at least Java 6 JRE or later installed. Run 'java -version' to see your Java version.
2. Change into the AtlasStyler-${project.version} directory 
4. ./start_AtlasStyler.sh
